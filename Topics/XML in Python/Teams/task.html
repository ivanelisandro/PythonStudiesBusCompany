A text file you will work with will have the following structure:

<team name="Snow white">
    <members>
        <member name="Dopy"/>
        <member name="Bashful"/>
        <member name="Sneezy"/>
        <member name="Sleepy"/>
        <member name="Happy"/>
        <member name="Grumpy"/>
        <member name="Doc"/>
    </members>
</team>

Your task is to extract all team members' names and output them in the same order, separated by spaces. For the example above, the answer would be: Dopy Bashful Sneezy Sleepy Happy Grumpy Doc.
